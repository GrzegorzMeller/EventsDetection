# EventsDetection

This repository contains final project python scripts for Knowledge Extraction and Information Retrieval course at the Universiy of Milan. 
Project aims at detecting historical events from text. Two experiments were done, one on the HISTO dataset (https://github.com/dhfbk/Histo) and second on self-made dataset by retrieving Wikipedia articles.
All details are described in the attached pdf. Python scripts performed on HISTO dataset and NER model are available in the histo folder, simillarly all scripts perfromed on Wikipedia dataset are available in the dbpedia folder.
